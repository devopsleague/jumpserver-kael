
# Kael

Kael 是 JumpServer 连接 GPT 的组件，计划支持多种 GPT, 

Kael 前端 vue3，后端使用 Fastapi 实现， 名字来源 [Kael](https://www.dotafire.com/dota-2/guide/kael-1867)。

## 支持的 GPT

- [x] ChatGPT
- [ ] Bing
- [ ] Bard


## 安装

1.下载项目

```shell
git clone https://github.com/jumpserver/kael.git
```

2.编译应用
